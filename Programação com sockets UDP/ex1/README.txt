-Requisitos:

->Descrição:
    Programa que incia chat com outro usuário, e envia mensagens para ele.

->Como executar:
    pip install emoji
    python3 ex1_chat.py <->> Inicia chat (Precisa de um outro usuário para conversar, abrir 2 chats no mínimo)

->Bibliotecas usadas:
    socket <->> criar o socket
    os <->> para fechar o arquivo
    ex1_utils <->> tipos do exercicio 1 e funções para ajudar

->Exemplo de uso:
    Executar dois programas ao mesmo tempo e inserir os dados corretamente, caso precise dos comandos digitar "help"
